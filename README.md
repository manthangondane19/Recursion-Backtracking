# Recursion & Backtracking

A collection of recursion and backtracking problems solved in C++, covering both standard practice problems and LeetCode problems.

## Problems

| File | Problem | Type |
|------|---------|------|
| `subsets.cpp` | Generate all subsets of an array | LeetCode #78 |
| `subsetswithsumk.cpp` | Print all subsets with sum equal to K | Standard |
| `onesubsetwithsumk.cpp` | Check if any one subset with sum K exists | Standard |
| `numberofsetwithsumk.cpp` | Count subsets with sum equal to K | Standard |
| `partitioning.cpp` | Palindrome partitioning | LeetCode #131 |
| `permutation.cpp` | Generate all permutations of an array | LeetCode #46 |
| `combinationsum2.cpp` | Combination sum II | LeetCode #40 |
| `reversearray.cpp` | Reverse an array using recursion | Standard |

## Concepts Covered

- Recursion tree and backtracking pattern (choose → explore → unchoose)
- Subset generation
- Permutations
- Palindrome partitioning
- Combination problems with constraints

## Language

C++
